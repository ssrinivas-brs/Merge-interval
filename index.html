Merge interval

Merging intervals refers to the process of combining overlapping or adjacent intervals into a consolidated or simplified form. An interval, in this context, is a range of values defined by a start point and an end point. When working with sets of intervals, merging them can reduce redundancy and simplify data representation. For example, if you have intervals [1,3],[2,6],[8,10],[15,18] the second interval overlaps with the first one.
